## Requirements:


- PHP ^8.1.2
- Composer 2.4.4
- Node.js & npm
- Sqlite for local

## Project setup local

```shell
git clone https://github.com/getmentor/getmentor-backend.git
cd getmentor-backend
make setup
```

## Project start local

```shell
make start
```
